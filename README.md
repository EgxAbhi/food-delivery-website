🍔 FoodDash – Full Stack Food Ordering Platform

FoodDash is a modern full-stack food ordering web application built using the MERN stack. It allows users to browse menu items, manage carts, place orders, and securely pay online using Stripe. It also includes an admin dashboard for managing orders and menu items in real time.

🚀 Features
👤 User Features
i) Secure authentication (Login/Signup)
ii) Browse food items with interactive UI
iii) Add/remove items from cart
iv) Place orders seamlessly
v) Secure online payment via Stripe
vi) Order tracking and status updates

🛠️ Admin Features
i) Add, update, delete menu items
ii) View all user orders
iii) Update order status (Pending → Delivered)
iv) Real-time data updates

🧰 Tech Stack
